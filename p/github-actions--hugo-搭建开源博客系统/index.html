搭建步骤

1. 从原作者的代码仓库中使用模板构建项目

  • github.com/CaiJimmy/hugo…

​​

我们根据这个模板来构建博客

‍

2. 构建两个仓库

我们配置 2 个仓库

  1. 网站展示仓库（public）
  2. Hugo 源码仓库

展示仓库就是我们的 hildam.github.io，而源码仓库则是 private 的 hilda-blog

自动化流程

我们在本地写好博客后，push 到 hilda-blog 仓库后，触发 Actions 操作，将 Hugo 构建好后的 public 文件夹，同步到 hildam.github.io 仓库中

这样就可以一次 push，自动构建了！

‍

3. 配置好 Actions 文件

在原作者模板项目中，修改 deploy.yml 文件，着重在 Deploy 阶段增加目标仓库名字，和 Gtihub Token 信息即可：

            - name: Deploy 🚀
              uses: JamesIves/github-pages-deploy-action@v4
              with:
                  # 指定发布的仓库
                  repository-name: HildaM/hildam.github.io
                  token: ${{ secrets.PAGE_TOKEN }}
                  branch: main
                  folder: public
                  clean: true
                  single-commit: true
